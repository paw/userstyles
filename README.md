## Nekoweb Editor Custom Theme
#### Install: [Via Github](https://github.com/paw/userstyles/raw/refs/heads/main/nekoweb-editor-custom-theme.user.css) / [Userstyles.World Mirror](https://userstyles.world/style/23293/nekoweb-editor-custom-theme)

### Description
A userstyle for reskinning the Ace Editor used on [Nekoweb](https://nekoweb.org/) for free-tier users.
- Choose one of the 3 site editor themes to overwrite. It overwrites Light by default.
- Optionally set a custom font. Leave it default if you like the font chosen with the dropdown. Note: The Ace Editor can only handle Monospace fonts, if you use a non-monospace font the cursor positioning will break.
- You can customize all the colors and it is built to support work with both light and dark color schemes.

## Userstyles.world: Sidebar Nav Tweaks
#### Install: [Via Github](https://github.com/paw/userstyles/raw/refs/heads/main/userstyles-world-sidebar-nav-tweaks.user.css) / [Userstyles.World Mirror](https://userstyles.world/style/24021/userstyles-world-sidebar-nav-tweaks)

### Description
A responsive userstyle for Userstyles.world that lets you tweak some stuff for the newer, sidebar navigation layout. Enable "Sidebar Navigation" in your settings to get the newer layout.
Options:
- **Top Bar for Large Screen Sizes** brings back the fixed top bar navigation style for large screensizes when enabled. **Top Bar Content Justify** lets you tweak where the nav will position stuff (most helpful for very large monitor sizes).
- **Move Sidebar to Right Side** allows you to reposition the sidebar to the right side of the screen. Works with Top Bar enabled as it only affects the smaller screen sizes.
- Finally, you can set custom values for the **Maximum Layout Width**, **Sidebar Width** (does not affect the top bar, but will affect the sidebar when the window is small enough), and the **Minimum Card Width** (default is now set to 250px from 300px so the intended 4 cards/row of the default layout on larger screen sizes will work).
- Compatible with custom color themes such as [Userstyles Tokyo Night by Josh](https://userstyles.world/style/16469/userstyles-tokyo-night-tokyonight)
